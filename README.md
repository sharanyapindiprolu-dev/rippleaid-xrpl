# RippleAid+ (XRPL)

**Tagline:** Verified Aid Vouchers with Enforced Eligibility (XRPL DID + Authorized Trust Lines)

## Overview
RippleAid+ prevents fraud in aid distribution using XRPL.  
Only verified recipients can receive and spend vouchers.  
Merchants get instant settlement and NGOs get full transparency.
